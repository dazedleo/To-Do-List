# To-Do List App

A simple and efficient **To-Do List application** built to help users manage daily tasks with ease.  
This project demonstrates clean design, CRUD operations, tests, and organized code architecture.

---

## Features

-  Signup  
-  Login
-  Add new tasks  
-  Edit existing tasks  
-  Delete tasks  
-  Persistent storage (localStorage / backend API)  
-  Responsive design  
-  Clean and modern UI  

---


---

##  Tech Stack

- **Frontend:** HTML / CSS / JavaScript  
- **Backend:** Django Rest Framework.  
- **Database:** SQLite 

---

##  Installation

```bash
# Clone the repository
git clone https://github.com/dazedleo/To-Do-List.git

# Navigate into the project
cd to_do_list

# Install dependencies
py manage.py -r requirements.txt

# Run the development server
py manage.py runserver

# Run the Tests
pytest
